# Processed RAG Data

Use this directory for cleaned and chunked RAG content in a later phase.

Examples:

- normalized JSON documents
- chunked text files
- metadata-enriched records

Preferred structured record fields:

- `id`
- `crop`
- `disease`
- `summary`
- `symptoms`
- `causes`
- `treatment`
- `precautions`
- `prevention`
- `severity`
- `source`
- optional `aliases`
- optional `tags`

Template:

- `disease_document.example.json`
